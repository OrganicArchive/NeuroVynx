# Documentation Archive

## Purpose

This archive contains the legacy development specifications, implementation plans, and session summaries that guided the early evolution of the project. These files are retained for historical reference and context, documenting the transition from the "NeuroSight" prototype phase to the current "NeuroVynx" framework.

> [!IMPORTANT]
> These documents represent legacy logic and should not be used as a reference for the current system architecture.

## Contents
- **Numbered Specs (0-9)**: Original architectural and requirement specifications.
- `Full_Summary.md`: Integrated summary of early project goals.
- `TESTING.md`: Historical testing guidance (partially migrated to `docs/ARCHITECTURE.md`).
- `LEGACY_SESSION_UPDATES.md`: Original raw development logs.

For current, framework-ready documentation, please refer to the files in the root `docs/` directory.
